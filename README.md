# Pixel_Art_Maker
Pixel Art Maker is a very simple way of using small pixels to create and design any shape that is desired. 
You are able to choose the grid size, as well as color.
Feel free to be as creative as possible when creating your masterpiece!
# Installation
1. Download the zip folder.
2. Open with code editer of choice.
3. Open HTML in any of the various web browsers.
