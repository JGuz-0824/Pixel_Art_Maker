# Pixel_Art_Maker

Pixel Art Maker is a very simple way of using small pixels to create and design any shape that is desired. 
You are able to choose the grid size, as well as color.
Feel free to be as creative as possible when creating your masterpiece!

## Installation

- **Clone** all files.
- Open with **code editer** of choice.
- Open HTML in any of the various **web browsers**.

## How to use

As you load the page in a web browser, there are a series of options before you can begin painting:

1. Choose the `height` and `width` of your canvas.
2. Choose the desired color to fill in the pixel grid(_canvas_).
3. Now begin painting, by filling in any pixel within your chosen grid size. You can change color on every pixel, or have different colors in the pixels. 

Click `submit` to reset your grid at the same size, or feel free to change the size at any time.
